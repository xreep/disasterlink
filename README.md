# 🌍 DisasterLink – Real-Time Disaster Coordination Platform

## Overview

DisasterLink is a real-time coordination platform designed to connect disaster victims, volunteers, and emergency coordinators through a unified system.

It focuses on accessibility and real-time communication, ensuring help can be requested and delivered even in low-connectivity environments.

---

## 🚀 Features

* Real-time request and response system
* Multi-channel access (web, missed call, emergency APIs)
* Live dashboards for coordinators and volunteers
* Scalable resource and request management
* Designed for low-connectivity and rural scenarios

---

## 🧠 System Design

The platform is built around real-time data synchronization:

* Victims can raise requests through multiple entry points
* Requests are instantly synced across dashboards
* Volunteers and coordinators receive live updates
* Resources are dynamically allocated based on availability

---

## 🛠 Tech Stack

* Frontend: React.js, Tailwind CSS
* Backend: Supabase (PostgreSQL, Realtime, Auth)
* APIs: Emergency services integration
* Deployment: Netlify
